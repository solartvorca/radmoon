# 🔔 Система Напоминаний - Инструкция

## Что было добавлено

### 1. **Настройки уведомлений для пользователей**
- **URL:** `http://127.0.0.1:5000/notification-settings`
- Пользователи могут выбрать способ получения напоминаний:
  - 📧 Email напоминания
  - 💬 Сообщения в ВКонтакте
  - ⏰ Время отправки напоминаний (по умолчанию 09:00)
- Есть кнопка "Тест" для проверки работы уведомлений

### 2. **Рассылка сообщений от администратора**
- **Доступ:** Админ портал → Раздел "Рассылка"
- Администратор может отправить сообщение всем пользователям, подписанным на уведомления
- Поля для заполнения:
  - Заголовок рассылки
  - Тема Email (поддерживает HTML)
  - Текст Email
  - Сообщение для ВК

### 3. **Автоматические напоминания**
- Система будет автоматически отправлять напоминания в указанное время
- Используется APScheduler для планирования

## Как использовать

### Для пользователя:
1. Перейди на страницу задания дня
2. Нажми на "Notification Settings"
3. Включи нужные способы уведомлений
4. Укажи свой VK ID (если выбрал уведомления в ВК)
5. Выбери время получения напоминаний
6. Нажми "Сохранить"
7. Опционально: нажми "Тест" для проверки

### Для администратора:
1. Войди в админ портал (`/admin-secret-portal`)
2. Прокрути вниз до раздела "Рассылка"
3. Заполни поля:
   - Тему письма
   - Текст письма (можешь использовать HTML теги)
   - Сообщение для ВК
4. Нажми "Отправить рассылку"

## Конфигурация

### Переменные окружения в `baza.env`:
```
VK_TOKEN=your_vk_token_here
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASSWORD=your_app_password
```

### Для Gmail:
1. Включи 2-фактор аутентификацию
2. Создай App Password (не обычный пароль)
3. Используй App Password в SMTP_PASSWORD

## Таблицы БД

### NotificationSettings
- user_id (FK)
- email_notifications (Boolean)
- vk_notifications (Boolean)
- vk_user_id (String)
- notification_time (String, format: "HH:MM")

### AdminMessage
- id (Integer PK)
- title (String)
- email_subject (String)
- email_body (Text)
- vk_message (Text)
- created_at (DateTime)
- sent_at (DateTime)
- recipient_count (Integer)

## API Endpoints

### Пользовательские:
- `GET /notification-settings` - Показать форму настроек
- `POST /notification-settings` - Сохранить настройки (JSON)
- `POST /test-notification` - Отправить тестовое письмо/сообщение

### Администраторские:
- `POST /admin-secret-portal/send-broadcast` - Отправить рассылку (JSON)

## Примеры JSON запросов

### Сохранение настроек:
```json
{
  "email_notifications": true,
  "vk_notifications": true,
  "vk_user_id": "123456789",
  "notification_time": "09:00"
}
```

### Отправка рассылки:
```json
{
  "title": "Напоминание о дне 5",
  "email_subject": "День 5 марафона",
  "email_body": "<h2>Здравствуй!</h2><p>Не забудь выполнить задание!</p>",
  "vk_message": "День 5 марафона - выполни задание и отправь отчет!"

```

## Решение проблем

### Email не отправляется:
- Проверь SMTP_HOST и SMTP_PORT в baza.env
- Убедись что используешь App Password для Gmail (не обычный пароль)
- Проверь что в консоли Flask нет ошибок

### ВК сообщения не отправляются:
- Проверь VK_TOKEN в baza.env
- Убедись что VK ID пользователя указан правильно
- VK ID должен быть число (например: 123456789)
